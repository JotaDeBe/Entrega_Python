# Proyecto de Katas en Python — Lógica y Programación

Este proyecto recopila una serie de 41 ejercicios de lógica, programación y estructuras básicas en Python, desarrollados con el objetivo de reforzar habilidades esenciales en el lenguaje: manejo de listas, funciones, excepciones, funciones lambda, estructuras de control, clases y programación modular.

Cada ejercicio ha sido resuelto aplicando buenas prácticas de código, uso adecuado de condicionales, comprensión de estructuras de datos, pensamiento computacional y escritura de funciones limpias y reutilizables.

Como estudiante en proceso de aprendizaje, este proyecto refleja la consolidación de conocimientos clave para un perfil Python Junior, mostrando capacidad para resolver problemas reales mediante programación.

**Objetivos del proyecto** 

Practicar la lógica de programación mediante retos progresivos.

Aprender a trabajar con colecciones de datos (listas, tuplas, diccionarios).

Usar funciones (map, filter, reduce) de forma práctica.

Desarrollar funciones personalizadas manejando parámetros opcionales.

Gestionar errores y excepciones correctamente.

Implementar clases en Python y aplicar POO básica.

Realizar pequeños programas interactivos que solicitan datos al usuario.

**Contenido del proyecto**

Los ejercicios se dividen en distintas categorías para cubrir varios aspectos del lenguaje:

  1. Manipulación de cadenas y textos

Frecuencias de letras

Procesamiento de palabras (buscar, eliminar, reemplazar)

Evaluación de frases

Anagramas

Enmascaramiento de cadenas

Longitud de palabras (map)

  2. Listas, números y operaciones con funciones funcionales

Ejercicios usando:

map() → procesamiento de listas

filter() → filtrado de elementos

reduce() → acumulación o reducción de datos

Incluye retos como:

Duplicar valores

Filtrar mascotas prohibidas

Calcular promedios

Obtener el producto total

Convertir dígitos a número

Concatenar palabras

Diferencias de valores

Extraer solo enteros de listas mixtas

  3. Manejo de excepciones

Se incluyen ejercicios prácticos como:

División con control de errores

Validación de edades

Listas vacías

Búsqueda de nombres con excepciones personalizadas

Estos ejercicios refuerzan una práctica esencial para programar de forma segura y profesional.

  4. Programación orientada a objetos (POO)

Dos ejercicios clave ponen en práctica clases:

✔ Clase Arbol

Implementación de atributos internos (tronco, ramas)

Métodos para manipular el árbol (crecer, añadir, quitar)

Devolver información detallada del objeto

✔ Clase UsuarioBanco

Atributos: nombre, saldo, tipo de cuenta

Métodos de operación bancaria:

retirar dinero

transferir entre usuarios

agregar saldo

Manejo de errores y validaciones

Estos ejercicios consolidan la base para entender objetos, estados y comportamiento en Python.

  5. Condicionales y lógica aplicada

Ejercicios como:

Identificar día, noche o tarde según la hora

Determinar calificaciones según nota

Cálculo de precio final aplicando descuentos

Permiten usar condicionales (if, elif, else) en escenarios prácticos.

 **Tecnologías utilizadas**

Python 3.x

Funciones integradas (map, filter, reduce)

Módulo functools para reducción de valores

Programación estructurada y POO

Manejo de excepciones (try / except)
